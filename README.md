# Task Manager (Java Console App)

Description
This is a simple console-based Task Manager written in Java.  
It allows users to manage a list of tasks using basic CRUD operations.

---

Features
- Add new tasks
- View all tasks
- Mark tasks as DONE
- Delete tasks
- Simple input validation

---

Technologies
- Java
- OOP (Classes & Objects)
- ArrayList
- Scanner (console input)

---

How to run
1. Open the project in IntelliJ IDEA
2. Run `Main.java`
3. Use the console menu to interact
